# Docker Compose
Suba com:
```
docker compose up -d --build
docker compose ps
docker compose logs -f app
```
## Endpoints

- GET, POST:
```
http://localhost:8080/api/users
```

- GET, PUT, DELETE:
```
http://localhost:8080/api/users/{id}
```

Swagger: 
```
http://localhost:8080/swagger-ui.html
```
